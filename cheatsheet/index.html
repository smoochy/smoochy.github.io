Welcome to my HTML Cheat Sheet

This Cheat Sheet can be used to quickly remember different HTML tags

Document Structure

These HTML tags define the structure of the document

HTML Tag Name Description
<html> HTML Defines the root of an HTML document
<head> Head Contains meta-information about the HTML document
<title> Title Specifies the title of the HTML document
<body> Body Contains the content of the HTML document

Text Formatting

These HTML tags define specific text styles

HTML Tag Name Description
<h1> - <h6> Heading 1 - Heading 6 Defines the headers (headings) of different levels
<p> Paragraph Defines a paragraph
<strong> Strong Represents strong importance or bold text
<title>em Emphasis Represents emphasized text or italic text
<u> Underline Represents underlined text

Lists

These HTML tags define how to create lists

HTML Tag Name Description
<ul> Bulleted List Creates an unordered (bulleted) list
<ol> Numbered List Creates an ordered (numbered) list
<li> List Item Defines a list item

Links & Navigation

These tags are used to link different things and define a navigation

HTML Tag Name Description
<a> Hyperlink Defines a hyperlink
<nav> Navigation Defines a navigation bar
